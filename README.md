# Geobacter Pattern Simulation

A WebGL-based simulation modeling emergent pattern formation by *Geobacter metallireducens* bacteria colonies interacting with iron minerals. This project aims to reproduce "leopard spot" patterns analogous to those observed in Martian rocks at Jezero Crater.

## Tech Stack

- 100% client-side web app
- **Svelte + Vite** - UI framework
- **TypeScript** - Application code
- **WebGL/three.js** - GPU-accelerated simulation

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

- `docs/` - Mathematical model documentation
- `src/` - Application source code
